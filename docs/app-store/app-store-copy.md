## Description

```text
Turn Timer keeps game nights, routines, and group activities moving with colorful visual countdowns. Create reusable rounds for players, recipe steps, classroom stations, meeting speakers, plant watering zones, and more.

KEEP TURNS FAIR

Set up a sequence once, then run it whenever you need it. Each round gets a name, color, sound, and duration so everyone can see whose turn or step is active.

BUILT FOR REAL LIFE

- Start a quick visual timer with one tap
- Use starter templates for game night, recipes, plants, classrooms, and meetings
- Build custom templates with named rounds and repeats
- Skip, restart, pause, do over, or add a round during a session
- Review recent session history after a timer completes
- Import and export portable Turn Timer template files
- Mark a favorite template and start it from widgets
- Use the Apple Watch app for simple wrist-based countdowns

SAVE THE SETUPS YOU REUSE

Turn Timer is strongest when you have a timer path you run again and again: a board game turn order, a recipe process, a class rotation, a meeting agenda, a workout circuit, or a watering routine.

TURN TIMER PRO

Turn Timer is free for quick timers, starter templates, basic playback, recent history, and one custom saved template. Turn Timer Pro unlocks unlimited saved templates, full history export, iCloud template sync, sharing, and favorite-template widgets with a one-time purchase.

PRIVATE BY DESIGN

Turn Timer does not use third-party tracking, ads, or analytics SDKs. Your templates and history stay on your device unless you use iCloud template sync through your Apple account.

Download Turn Timer and make time visible for the people, steps, and routines that matter.

Questions or feedback? Contact: [support email]
```

## Review Notes Draft

```text
Turn Timer is a visual sequence timer. The core timer, starter templates, and one custom saved template are free. The non-consumable Turn Timer Pro unlock uses product ID turntimer.pro.unlock and unlocks unlimited saved templates, full history export, iCloud template sync, sharing/import/export workflows, and favorite-template widgets.

iCloud sync uses the user's private CloudKit database under container iCloud.Dan.Visual-Timer. Widgets use App Group group.Dan.Visual-Timer for compact favorite-template snapshots and pending start requests. Widgets do not run a background live timer; tapping Start opens the app and starts the selected template.
```
